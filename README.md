# L'ORACLE

formulaire de contact avec des bruit et il est rigolo


### Prérequis

- Node.js (version 18 ou supérieure)
- npm ou yarn

### Étapes

1. Cloner le repository :
  ```bash
  git clone https://github.com/Ashokaas/communiquer_avec_oracle
  cd oracle-site
  ```

2. Installer les dépendances :
  ```bash
  npm install
  ```

3. Lancer le serveur de développement :
  ```bash
  npm run dev
  ```

4. Ouvrir `http://localhost:3000` dans votre navigateur
